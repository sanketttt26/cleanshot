# Framy - Screenshot Background Extension

A Chrome browser extension that captures screenshots and adds beautiful backgrounds with one click. Perfect for social media, blogs, and documentation.

![Framy Extension](icons/icon128.png)

## Features

### Screenshot Capture
- **Visible Tab** - Capture what's currently visible
- **Select Area** - Click and drag to capture a specific region

### Background Enhancement
- **Gradients** - Multiple preset gradients and custom gradient editor
- **Solid Colors** - Pick any color
- **Custom Image** - Upload your own background image

### Annotation Toolbar
- **Drawing Tools** - Arrow, Line, Circle, and Rectangle
- **Color Picker** - HSL color slider for annotations
- **Undo/Redo** - Keyboard shortcuts (Ctrl+Z, Ctrl+Y) for annotation history

### Customization
- Padding size slider (20-150px)
- Corner radius control (0-40px)
- Drop shadow toggle
- Custom gradient editor with multiple color stops

### Export
- PNG, JPG, WebP formats
- Copy to clipboard
- Direct download

## Installation

1. Open Chrome and navigate to `chrome://extensions`
2. Enable **Developer mode** (toggle in top-right corner)
3. Click **Load unpacked** and select the `Framy` folder

**Keyboard Shortcut:** Alt + Shift + S (quick capture visible tab)

**Context Menu:** Right-click on any page â†’ **Framy Screenshot** â†’ Choose capture type

## Usage

1. Click the Framy icon in your browser toolbar
2. Choose capture type: **Visible Tab** or **Select Area**
3. Customize background, padding, corner radius, and shadow
4. Use the annotation toolbar to draw arrows, lines, circles, or rectangles on your screenshot
5. Select export format (PNG, JPG, WebP)
6. Click **Download** or **Copy** to clipboard
